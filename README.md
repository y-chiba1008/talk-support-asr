# 会話サポートアプリ (Talk Support ASR)

「会話サポートアプリ」は、音声を認識し、テキスト化（文字起こし）と会話の分析を行うことで、円滑なコミュニケーションを支援するアプリケーションです。

## 主な機能＆ポイント

- **音声認識**: 録音停止後にサーバーへ音声を送信し、テキストに変換します。
- **認識結果の表示**: 直前の録音内容をテキストとして画面に表示します。
- **音量レベル表示**: 入力音声のレベルをリアルタイムで確認できるボリュームバーを搭載。

## 使い方

1. **アプリの起動**: 公開されているURLにアクセスします。
2. **マイクの許可**: ブラウザからマイクの使用許可を求められたら「許可」を選択してください。
3. **録音開始**: 画面上のマイクボタン（録音ボタン）をクリックすると、録音が開始されます。
4. **会話**: マイクに向かって話します。
5. **録音停止・認識**: 再度マイクボタンをクリックすると録音が停止し、サーバーで解析されたテキストが表示されます。

## システム構成

本プロジェクトは以下の構成で成り立っています：

- **Frontend**: React + Chakra UI を使用した、直感的でモダンなユーザーインターフェース。
- **Backend**: FastAPI を使用した高性能な API サーバー。Whisper 等のモデルを利用した音声解析を行います。
- **Labeling Tool**: 音声データの精度向上のためのアノテーション環境（Label Studio）。
